# weather_app

Build a weather app using the openWeatherMap weather API.

## Technologies to build with
- HTML
- CSS
- JavaScript

## API Link
https://openweathermap.org/price

## How to use it:
- create an account with openWeather map by selecting the free plan.
- get your API key
- use the API with JavaScript and build a weather app that takes in 2 inputs: the country, and city.
- finally display the result under when the API returns the result
